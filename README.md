# main-mjgame-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面各自实现特定的功能或展示，彼此之间没有强关联，也不针对任何特定域名或网站。

## 项目简介

这是一个资源归档性质的仓库，主要存放各类独立的 HTML 文件。每个文件都可以在浏览器中直接运行或预览，适合作为前端小工具、演示页面、交互原型或静态内容展示的集合。

## 目录说明

仓库根目录下直接存放 HTML 文件，部分资源可能按功能或用途放在子文件夹中。目前目录结构如下：

```
├── index.html          # 仓库入口页面（如有）
├── page1.html          # 示例页面1
├── page2.html          # 示例页面2
├── folder_name/        # 分类文件夹
│   ├── sub_page.html
│   └── assets/         # 页面依赖的资源文件
└── README.md           # 本文件
```

具体文件列表请直接浏览仓库文件。

## 页面归档说明

- 每个 HTML 页面的功能、使用方法或注意事项，通常会在文件头部注释或页面内说明。
- 页面之间相互独立，互不依赖。所有页面均使用标准 HTML/CSS/JavaScript 编写，部分可能引用外部 CDN 资源。
- 归档的页面可能包含实验性、临时性或已完成使命的内容，不保证长期更新或维护。

## 维护说明

- 本仓库主要作为静态资源存放地，不定期更新或新增页面。
- 不提供持续集成、自动化部署或版本管理以外的额外服务。
- 如有页面无法正常访问或出现错误，可提交 Issue 反馈，但不保证及时修复。
- 欢迎提交 Pull Request 修正错误或补充有用页面，但请保持页面独立性和简洁性。

## 使用方式

1. 直接点击仓库中的 HTML 文件，使用浏览器预览（GitHub 会自动渲染部分页面）。
2. 或克隆仓库到本地，用浏览器打开相应文件。
3. 如需部署到服务器，直接托管整个仓库内容即可。
